# 統計情報ガイド

プロジェクト全体のコード行数・ファイル数・バイト数を確認する方法を説明します。

---

## 使用スクリプト

`scripts/stats.mjs`

---

## 実行方法

プロジェクトルートで実行します。

### テキスト表（デフォルト）

```sh
node scripts/stats.mjs
```

出力例：

---

### Markdown 形式で出力

```sh
node scripts/stats.mjs --markdown
```

出力をそのまま Markdown ドキュメントに貼り付けられる表形式で出力します。

---

### JSON 形式で出力

```sh
node scripts/stats.mjs --json
```

ツールやスクリプトからデータを利用したい場合に使います。

---

### ファイルに保存

```sh
node scripts/stats.mjs > docs/stats-output.txt
node scripts/stats.mjs --markdown > docs/stats.md
node scripts/stats.mjs --json > docs/stats.json
```

## 集計対象

| 対象ディレクトリ | 対象拡張子 | 内容 |
|---|---|---|
| `src/app` | `.ts` `.tsx` `.css` | Next.js アプリエントリ |
| `src/game/core` | `.ts` | コアロジック（迷路・戦闘・AI 等） |
| `src/game/systems` | `.ts` | 各種システム（音声・描画・入力） |
| `src/game/ui` | `.tsx` `.ts` | React UI コンポーネント |
| `src/game/assets/data` | `.json` | ゲームデータ JSON |
| `scripts` | `.ts` `.mjs` `.js` | 生成・補助スクリプト |
| `tests` | `.ts` `.js` | テストコード |
| `docs (HTML)` | `.html` | ドキュメント HTML（直下のみ） |
| `docs (MD)` | `.md` | ドキュメント Markdown（再帰） |
| `public/sprites (PNG)` | `.png` | ドット絵スプライト（行数は 0 として集計） |

> PNG 等のバイナリファイルはバイト数のみ計測し、行数は 0 として扱います。

---

## 注意事項

- `node_modules/` や `.next/` は集計対象外です。
- `public/sprites (PNG)` の行数は常に 0 です（バイナリのため）。
- スクリプト自体 (`scripts/stats.mjs`) も `scripts` カテゴリに含まれます。
